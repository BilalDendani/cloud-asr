CloudASR CrowdFlower Audio Transcription

Thank you for deciding to help us annotate recordings. In the following text we will walk you through creation of the CrowdFlower task for audio transcription.

  1. Login to your CrowdFlower account as a customer.
  2. Click on the Create Job button.
  3. Click on the View All Templates, select Audio Transcription and click on the Use This Template button.
  4. On the Build your job page leave everything as it is and continue by clicking on the Save button.
  5. Go to the Data page and upload there this file. In this file there are 100 selected recordings that we need to transcribe.
  6. Go to the Job Settings page and set the following values:
    Performance Medium value
    Jugments per row: 1
    Number of rows displayed on each page: 1
    Payment amount per page 5
  7. Now, you can start the task on Launch page.
  8. Finally, when the task is finished. Please, upload the csv file with results here.
